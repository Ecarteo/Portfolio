# Personal Portfolio Website

![Project Preview Thumbnail](.github/preview.jpg)

View Live Site: [mateocalderon.com](https://mateocalderon.com/)

This repository contains the source code for my personal portfolio website, which showcases my software development projects, technical skills, and professional journey. 

The site is built as a responsive, clean, and modern single-page application designed to give recruiters and collaborators a direct look at my capabilities.

---

## 🔴 Tech Stack & Infrastructure

*   **Frontend:** HTML5, CSS3, JavaScript.
*   **Hosting:** GitHub Pages.
*   **Domain:** Custom domain with configured DNS records.

---

## 🟠 Key Features

*   **Responsive Layout:** Fully optimized for desktop and mobile devices.
*   **Project Gallery:** An interactive showcase linking directly to live deployment and source code.
*   **CI/CD Deployment:** Automatically deploys and updates via GitHub Pages on every push to the `main` branch.
